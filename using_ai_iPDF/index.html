Quick Start Guide

1. Installation

# Install dependencies
pip install -r requirements_improved.txt

# Set up environment variables
export GROQ_API_KEY="your_api_key"

# Run the application
streamlit run ipdf_improved.py

2. Usage

  • Upload PDFs: Use the sidebar to upload one or more PDF files
  • Process Documents: Click "Process Documents" to extract and index content
  • Ask Questions: Use the chat interface to query your documents
  • View Analysis: Check the Analysis tab for summaries and statistics

3. Configuration Options

  • LLM Selection: Choose between Groq (cloud) or Ollama (local)
  • OCR Fallback: Enable for scanned documents
  • Confidence Thresholds: Adjust quality requirements
  • Chunking Strategy: Optimize for your document types
